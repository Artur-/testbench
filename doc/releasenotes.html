Release Notes for Vaadin TestBench ${project.version}

Overview of Vaadin TestBench 3

Vaadin TestBench is a UI browser automation tool used for automated testing of Vaadin UIs in the browser. Vaadin TestBench is based on Selenium 2 and provides a WebDriver based API for controlling different browsers on multiple platforms.

Changes Since 3.0

  • Headless testing support using PhantomJS
  • UX improvements

Package Contents

The package contains the following files and directories: 

documentation/
 The documentation folder contains the TestBench library API documentation, a PDF excerpt of this chapter of Book of Vaadin, and the license. 
example/
 The example folder contains a Maven example project with examples of how to use Vaadin TestBench. These include the project descriptor and a Vaadin test project, as well as the JUnit test Java source files. 
maven/
 The Maven folder contains a version of the Vaadin TestBench libraries that you can install in your local Maven repository along with short instructions on how to install. You can also access these from the Vaadin add-ons maven repository. 
vaadin-testbench-recorder-${project.version}.xpi
 This is the Vaadin TestBench Recorder, which you can install in Firefox. Please see the manual for installation instructions. 
vaadin-testbench-standalone-${project.version}.jar
 This is the Vaadin TestBench library. It is a standalone library that includes the Selenium 2 WebDriver along with all other dependencies.

Migration Notes

Vaadin TestBench 2.x tests cannot directly be converted to run on TestBench 3. For existing projects, it is recommended to continue using TestBench 2 for existing tests and installing TestBench 3 in parallel if you want the new features. For new projects it is recommended to use only TestBench 3.

Supported Platforms and Browsers

Vaadin TestBench 3 fully supports Windows, Mac OS X and Linux as well as iOS and Android for running tests. The following browsers are supported:

  • Google Chrome 13+
  • Internet Explorer 6+
  • Mozilla Firefox 3+
  • Opera 11.5+
  • Mobile browsers:
    • Android 2.3+
    • iOS 4+

Known issues

The following issues are known for this release:

  • Drag'n'Drop: Replay of Drag'n'Drop using the WebDriver API with the FirefoxDriver does not work if dragging is initiated from a Tree. Tables work fine.
  • RichTextArea: Playback of a recording from TestBench Recorder doesn't work, but playback using the WebDriver API works.
